# Installation

## From the Obsidian Plugin Manager

Install through the Obsidian plugin manager: [https://obsidian.md/plugins?id=tasks-map](https://obsidian.md/plugins?id=tasks-map)

## Local Installation

1. **Clone or Download:** Place the plugin folder in your vault's `.obsidian/plugins/tasks-map/` directory.
2. **Install Dependencies:**
   ```sh
   npm install
   ```
3. **Build the Plugin:**
   ```sh
   npm run build
   ```
4. **Enable in Obsidian:**
    - Open Obsidian settings → Community plugins → Enable "Tasks Map".
