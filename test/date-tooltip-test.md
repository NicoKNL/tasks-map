## 实现的技术要点

1. **日期提取**：使用 `extractDateFromTaskText()` 函数从任务文本中提取日期
2. **格式转换**：将 YYYY-MM-DD 格式转换为 YYYYMMDD
3. **相对日期处理**：支持 today/tomorrow/yesterday
4. **悬停检测**：通过 React 状态管理鼠标悬停
5. **位置计算**：工具提示显示在节点上方，垂直偏移可配置
